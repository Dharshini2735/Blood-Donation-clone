# RedLife - Blood Donation Website (Figma Clone)

## 📌 About the Project

RedLife is a responsive Blood Donation website UI developed as a training project by implementing a Figma design. The objective of this project was to practice front-end development and improve HTML and CSS skills by recreating the given design as accurately as possible.

## ✨ Features

* Responsive design for Mobile, Tablet, and Desktop
* Hero section with donor search panel
* Blood donor registration form
* Blood donor and blood bank search section
* About Us and Core Values sections
* Statistics cards and information cards
* Responsive navigation and footer
* Font Awesome icons integration

## 🛠️ Technologies Used

* HTML5
* CSS3
* Font Awesome

## 📱 Responsive Design

The website is optimized for:

* Mobile Devices
* Tablets
* Laptops/Desktop Screens

## 🎯 Purpose

This project was created as part of my training to practice converting a Figma design into a fully responsive web page using only HTML and CSS.

> **Note:** This is a front-end UI clone created for learning purposes. It does not include backend functionality or database integration.
